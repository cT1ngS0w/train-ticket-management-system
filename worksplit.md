# 火车票务管理系统 – 任务分配

大家好！由于我们小组共有5名成员，我希望每位成员都能选择**一个开发模块**。我将负责项目管理、数据库设计、系统集成以及 GitHub 仓库管理，同时会有另一名成员主要负责软件测试与文档编写。

请大家仔细阅读下方的三个编程模块，并回复告知你的首选模块。如果有多人选择了同一个模块，我们将进行讨论与调整。

---

### 团队协作规范

* 我们将使用 **GitHub** 进行团队协作。
* 每位开发人员将在自己的 **功能分支（feature branch）** 上进行开发。
* 代码变更需要通过 **拉取请求（Pull Requests）** 来提交合并。
* 未经讨论沟通，请尽量避免修改其他成员负责的模块。
* 我们将定期同步开发进度，并在整个项目周期中持续进行模块集成。

请回复告知你首选的模块（A、B 或 C）。如果可能的话，也可以附上你的第二选择，以防出现多人选择同一模块的情况。
